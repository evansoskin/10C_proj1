ReadMe file for PIC10C Project 1

This project is based on the game siete y medio, and is an assignment to make use of git and test our knowledge of verison control software.
The game itself is similiar to Blackjack, the goal is to get to 7.5 card value without going over using only cards Ace, 2-7, Jack, Queen, and King with Ace's value being 1 and Jack's, Queen's, and King's values all being equal to 0.5.
